# Funnel Rules

One funnel, deliberately ordered, on every page.

## The path

```
Campaigns → Case Studies → Audience → Process → Work With Us
```

The logic: understand the offer → see it worked → verify the reach is real → trust the system → start. Navigation lists pages in this order; in-page links push forward along it.

## Page jobs

| Page | Job | Pushes toward |
|---|---|---|
| `/` | Make the whole argument in miniature | `/campaigns` (and every preview's page) |
| `/campaigns` | Sell the mechanism | `/case-studies`, `/work-with-us` |
| `/case-studies` | Prove it worked | `/audience`, `/work-with-us` |
| `/audience` | Prove the reach is real | `/process`, `/work-with-us` |
| `/process` | Prove it's a system, reduce risk | `/work-with-us` |
| `/about` | Prove the person | `/work-with-us` |
| `/work-with-us` | Convert | email |

## CTA rules

- **One primary action site-wide:** Work With Us / email. Same label, same style, everywhere.
- Every page has the primary CTA at least top and bottom; long pages add one mid-page.
- At most one secondary CTA per section, always pointing forward along the funnel — never sideways, never outward.
- No dead ends: every page bottom has a next step. Footers carry a conversion block on every page.

## Email-first contact

- The conversion action is a `mailto:` with a prefilled, context-specific subject (e.g. "Campaign inquiry — [type]").
- `/work-with-us` includes a what-to-include checklist so the first email arrives qualified.
- No multi-step forms, no calendars gated behind forms, unless intake explicitly asks for them.

## Anti-patterns

- Multiple competing primary buttons on one screen
- CTAs that leave the funnel (social links styled like CTAs mid-page — socials live in the footer)
- "Learn More" links that go nowhere decisive
- Burying `/work-with-us` deeper than one click from anywhere

## Verification

Crawl the built site: every page reachable, every page has a forward link, every mailto resolves with the right subject, zero dead ends. Record as launch-review evidence.
